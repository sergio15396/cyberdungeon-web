# CyberDungeon

CyberDungeon es una web informativa del juego de exploración y combate por turnos ambientado en un entorno cibernético. En ella se presentan los personajes, mecánicas y sistema de combate del juego, además de permitir acceder directamente a la partida.

## Características
- Navegación por pestañas (Inicio, Personajes, Combate, Mecánicas)
- Diseño responsive con menú móvil
- Sistema interactivo para visualizar monstruos por nivel
- Acceso directo al juego

## Tecnologías
- HTML5
- CSS3
- JavaScript
